Project Goal

The goal of this project was to design and implement a responsive landing page using HTML, CSS and JavaScript, following modern UI/UX and accessibility principles.

The landing page simulates a real digital product and focuses on clear communication, usability, and lead generation.

Why Local Reputation & SEO

  • Local businesses depend on online reviews and visibility on maps/platforms.
  • A strong reputation helps attract new customers, build trust, and increase revenue.
  • This project shows how a landing page can communicate these benefits clearly.

Technology Stack

  • HTML
  • CSS
  • JavaScript
  • GitHub Pages (deployment)

The project does not rely on external frameworks or libraries.

Semantic HTML Structure

The page is structured using semantic HTML elements to improve accessibility, readability, and SEO.

  • Semantic sections: header, main, section, footer
  • Clear heading hierarchy (h1–h3)
  • Form labels and meaningful attributes

Responsive Layout

The layout adapts across desktop and mobile using CSS Grid, Flexbox, and media queries.

  • Grid-based sections for clear structure
  • Mobile navigation adapts to smaller screens
  • Readable typography and spacing

Navigation & Dropdown

Navigation includes a desktop dropdown (“Solutions”) and a mobile burger menu. Interactions are handled with JavaScript.

  • Dropdown open/close on click
  • Close on outside click and Escape
  • Mobile menu toggles via burger button

Call to Action Design

CTA buttons guide the user toward conversion with clear hierarchy: primary and secondary actions.

  • Primary: “Book a meeting”
  • Secondary: “Watch a demo”
  • Hover states for feedback

Modal Video Interaction

Clicking “Watch a demo” opens a modal window using the <dialog> element. It includes accessible close controls.

  • Open modal via button click
  • Close via button / outside click
  • Embedded video in iframe

Feature Cards

The feature section presents main benefits with a clear content hierarchy and supporting visuals.

  • Grow local rating
  • Client retention

Interactive Steps

The “How We Grow Your Business” section includes interactive steps. Clicking each step updates the content dynamically.

  • Step 1: Rating & visibility
  • Step 2: Feedback recovery
  • Step 3: Return customers

Analytics Section

A dedicated block shows how analytics and performance data can be presented in a clear and readable way.

Solutions by Industry

Solutions are grouped by industry to increase relevance for different users.

  • Restaurants, Clinics, Beauty
  • Hotels, Education, Retail

Lead Generation Form

The form includes required fields, accessible labels, and client-side validation using JavaScript.

  • Validation for name, email, selection, consent
  • Status messages for feedback

Cookie Consent Logic

The cookie banner demonstrates user choice handling and persistence using browser storage.

  • Accept / Decline actions
  • Banner hides after selection
  • Choice is remembered on reload

Code Organization

The project is organized into separate files for structure, styling, and behavior. This improves readability and maintenance.

  • index.html
  • css/styles.css
  • js/script.js
  • img/ assets

Deployment

The project is deployed using GitHub Pages, making it accessible as a live website.

Live demo: irina-kovalenko.github.io

Conclusion

This project demonstrates the ability to design a modern interface, write semantic HTML, implement interactive features with JavaScript, and deploy a complete web project.
